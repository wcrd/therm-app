Therm App enables engineers, architects, and contractors to model and analyse custom constructions.
Evaluate the thermal performance implications of materials, thickeness, air cavities, and ordering.

Drag & Drop builder

Easily create and re-order material layers in a drag and drop grid.

Import & Export to CSV

Export all created constructions to CSV for documentation or use in other software. Import saved constructions from CSV back into Therm App for visualisation and editing.

History & Library

Previously created constructions are stored in the History table for re-use. Constructions can be named, reloaded for editing or visualisation, or exported to CSV from the History table.

Temperature Plotting

Set internal and external temperature to view the temperature gradient through the construction. Identify possible condensation issues within the constructions.
